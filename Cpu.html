CPU BASICS

A CPU contains three parts:

  • The arithmetic/logic unit
    The control unit
    The register unit


  • The arithmetic/logic unit,contains the circuitry that performs operations on data.
  • The control unit, which contains the circuitry for coordinating the machine’s activities; and
  • The register unit contains data storage called registers, that are used for temporary storage of information within the CPU. Some of the registers in the register unit are general-purpose registers, while others are dedicated to a specific task.

    • General-purpose registers
    are used to temporarily store data that the CPU is manipulating. The inputs to the arithmetic/logic unit's circuits are held in these registers, which also serve as storage for the unit's results. The control unit transfers data from main memory into general-purpose registers, informs the arithmetic/logic unit which registers hold the data, activates the appropriate circuitry within the arithmetic/logic unit, and tells the arithmetic/logic unit which register should receive the result to perform an operation on data stored in main memory.

    A group of cables called a bus connects a machine's CPU and main memory for the purpose of exchanging bit patterns. The CPU extracts data from main memory through this bus by sending the address of the relevant memory cell together with an electrical signal instructing the memory circuitry to retrieve the data in that cell. The CPU stores (writes) data in memory in a similar way by supplying the address of the target cell and the data to be stored, as well as the proper electrical signal instructing main memory to store the data being transferred to it.


    If the control unit is programmed to extract the program from memory, decode the instructions, and execute them, the machine's program may be updated by simply altering the contents of the computer's memory rather than rewiring the CPU. The concept of keeping a computer's software in its main memory is known as the stored-program concept, and it is now the industry standard..


    CPUs are intended to identify instructions encoded as bit patterns in order to apply the stored-program notion. The machine language refers to the set of instructions as well as the encoding mechanism. A machine-level instruction (or, more generally, a machine instruction) is an instruction written in this language..
